## 🧠 교육 방법론

### 핵심 전략: "나선형 PBL + 역피라미드"

```
[라운드 1~3]  기초 체력        → 파이썬, 수학, 데이터 다루기
[라운드 4~6]  ML 핵심 무기     → 지도/비지도학습, 시계열, NLP
[라운드 7~8]  딥러닝 + 최적화  → CNN, RNN, Transformer, Convex Opt
[라운드 9]    HFT 시스템 설계  → 마이크로스트럭처, 저지연 아키텍처

```

### 라운드 8 — Convex Optimization + 포트폴리오 최적화 + Transformer

**목표:** Convex Optimization 이론을 이해하고 포트폴리오/전략 최적화에 적용한다

**선수지식:** 라운드 1~7 (특히 라운드 2 선형대수)

| 구분 | 내용 |
|------|------|
| Convex Optimization 이론 | 볼록집합, 볼록함수, KKT 조건, 라그랑주 승수법 |
| 최적화 알고리즘 | 경사하강법 변형(SGD, Adam), 뉴턴법, 내부점법 |
| CVXPY 실습 | 파이썬 Convex Optimization 라이브러리 실전 사용 |
| 포트폴리오 최적화 | Mean-Variance(마코위츠), Risk Parity, Black-Litterman |
| Transformer | Self-Attention, Positional Encoding, 금융 시계열 적용 |
| 교재 연동 | MLAT Ch.5 (전략 평가), Ch.20~21 (Attention/Transformer) |
|  | MLDSF Ch.7~8 (포트폴리오 최적화 케이스스터디) |
| 미니 프로젝트 | CVXPY로 샤프비율 최대화 포트폴리오 + Transformer 수익률 예측 |

### 라운드 10 — 통합 프로젝트: Convex Opt + DL 기반 HFT 시스템

**목표:** 전체 파이프라인을 통합하여 실전 HFT 시스템 프로토타입을 완성한다

**선수지식:** 라운드 1~9 전체

| 구분 | 내용 |
|------|------|
| 시스템 아키텍처 | 데이터 수집 → 피처 생성 → 모델 예측 → 최적화 → 주문 실행 |
| 딥러닝 시그널 | Transformer/LSTM 앙상블 예측 모델 |
| Convex Opt 실행 | 실시간 포지션 사이징 + 리스크 제약 최적화 |
| 백테스트 | Zipline/Backtrader로 전략 시뮬레이션 |
| 리스크 관리 | VaR, CVaR, 드로다운 제한, 포지션 한도 |
| 성과 평가 | 샤프비율, 소르티노비율, 최대낙폭, 승률 |
| 교재 연동 | MLAT Ch.5, 8 (백테스트, ML4T 워크플로우) |
|  | MLDSF 전체 복습 + 통합 |
| 최종 산출물 | 완전한 HFT 시스템 프로토타입 (코드 + 백테스트 리포트) |

별도 외부 자료를 찾아볼 필요 없이, 매 라운드 HTML이 올인원 교재 역할을 합니다.

---

## 🔄 라운드별 진행 방식 (매 대화)

```
1. 이전 라운드 복습 퀴즈 (2~3문제)
2. 이번 라운드 핵심 개념 HTML 강의 생성
   - 이론 설명 (수식 + 직관적 비유)
   - 파이썬 코드 예제 (복붙해서 바로 실행 가능)
   - 시각화 (차트, 다이어그램)
3. 교재 해당 챕터 핵심 요약
4. 실습 코드 파일 생성 (.py)
5. 미니 프로젝트 과제 제시
6. 다음 라운드 예고
```

### 핵심 규칙

- **한 채팅 = 한 라운드** — 매 대화마다 해당 라운드의 HTML 강의를 이어서 집필
- **이전 라운드 이어짐** — 각 라운드는 이전 내용을 기반으로 연결
- **3라운드마다 피드백** — R3, R6, R9 완료 시 학습 점검 + 난이도 조절 + 보충 필요 영역 확인
- **산출물** — 매 라운드마다 `round_XX/lecture_XX.html` + `practice_XX.py` 생성

### 피드백 세션 (R3, R6, R9) 포함 내용

1. 지난 3라운드 핵심 복습 퀴즈 (5~10문제)
2. 이해도 자가 체크리스트
3. 부족한 부분 보충 설명
4. 다음 3라운드 난이도 조절 여부 확인
5. 학습 페이스 점검 및 플랜 수정
